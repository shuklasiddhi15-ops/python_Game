# 🎯 Number Guessing Game

A fun and interactive **Number Guessing Game** built with **Python** and **Streamlit**.  
Guess the secret number before you run out of tries — with real-time hints, difficulty levels, and balloon animations on every game end!

---

## 🚀 Live Features

- 🎮 **3 Difficulty Levels** — Easy (1–50), Medium (1–100), Hard (1–200)
- 🔥 **Hot/Cold Hint System** — tells you if you're burning close or way off
- ⬆️⬇️ **High / Low Feedback** — instant direction after every guess
- 🧮 **Live Stats Card** — guesses made, tries left, max tries
- 📊 **Progress Bar** — visual indicator of how many tries you've used
- 🏷️ **Guess History Chips** — every guess logged with color-coded direction
- 🎈 **Balloon Animation** — pops on both win and loss
- 🔁 **Play Again** — restart instantly without refreshing

## 📦 Installation

**1. Clone the repository**
```bash
git clone https://github.com/your-username/number-guess-game.git
cd number-guess-game
```

**2. Install dependencies**
```bash
pip install streamlit
```

**3. Run the app**
```bash
streamlit run number_guess_game.py
```

---

## 🎮 How to Play

1. Pick a difficulty from the **sidebar**
2. Type your guess and hit **Guess! 🚀**
3. Follow the hints — too high, too low, or 🔥 burning hot!
4. Win before your tries run out
5. 🎈 Balloons pop whether you win or lose — no hard feelings!
